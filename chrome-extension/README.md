# Chrome Extension Setup

## Icon Files Required

The Chrome extension requires actual PNG icon files. Currently, the icon files are placeholders.

### Required Icon Files:
- `icons/icon16.png` - 16x16 pixels
- `icons/icon48.png` - 48x48 pixels  
- `icons/icon128.png` - 128x128 pixels

### To Fix:
1. Create or obtain proper PNG icon files
2. Replace the placeholder text files with actual PNG images
3. Ensure icons are properly sized (16x16, 48x48, 128x128)

### Icon Creation:
You can create icons using:
- Online icon generators
- Image editing software (GIMP, Photoshop)
- Icon design tools
- AI image generators

The icons should represent the Workday automation functionality and be professional in appearance.
